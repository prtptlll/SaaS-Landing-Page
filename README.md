#  Modern SaaS Landing Page

A clean, responsive SaaS landing page built with a focus on UI design, layout structure, and user experience.

## Features

* Hero section with strong call-to-action
* Fully responsive layout (mobile, tablet, desktop)
* Interactive pricing cards (selectable plans)
* Smooth navigation (CTA buttons scroll to sections)
* Features, testimonials, and FAQ sections
* Clean spacing, typography, and modern UI

##  Tech Stack

* HTML
* Tailwind CSS
* JavaScript

##  Purpose

This project demonstrates:

* Frontend layout structuring
* UI/UX design thinking
* Responsive design principles
* Basic interactivity using JavaScript
